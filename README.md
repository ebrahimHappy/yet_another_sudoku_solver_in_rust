# Yet Another Sudoku Solver in Rust
It is just a hubby project to learn rust.

## Build
```
git clone https://github.com/ebrahimHappy/yet_another_sudoku_solver_in_rust.git
cd yet_another_sudoku_solver_in_rust
rustc main.rs
```

## Run
```
./main
```
